# אשדודאנס 2026 - אפליקציית תוכנייה אישית

פרויקט Next.js המוכן לפריסה ב-Vercel. ראה את "מדריך-פרסום-Next.js.md" שקיבלת בשיחה עם קלוד להוראות מלאות.

## הרצה מקומית (אופציונלי, לבדיקה במחשב שלך)
```bash
npm install
npm run dev
```
ואז פותחים http://localhost:3000

## פריסה
1. מעלים את התיקייה הזו ל-repository חדש ב-GitHub
2. מתחברים ל-Vercel עם GitHub, בוחרים את ה-repository, לוחצים Deploy
3. זהו - מקבלים קישור חי
